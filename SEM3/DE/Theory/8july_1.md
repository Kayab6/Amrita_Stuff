# ANALOG VS DIGITAL
## ANALOG
An analog system processes analog signals, which are signals that change continuously with time.

The signal can have any value within a certain range (called the dynamic range)

Analog Signal

The graph looks smooth.

Voltage

Notice:

Smooth curve
No sudden jumps
Every point has a value
Dynamic Range

The slide says

"can take any value across a continuous range known as dynamic range."

This simply means:
Suppose a microphone produces voltages between

-5 V  -------------------- +5 V

The signal can be

1.2V
2.34V
-0.75V
4.999V

Anything in between.

That entire interval is called the dynamic range.


## Digital Devices and Systems
Definition

A digital system processes digital signals.

Digital signals have only discrete values instead of infinitely many values.

Most computers use only two values:

0
1

These are called binary digits (bits).

Digital Signal

Instead of a smooth wave, the signal jumps.

Voltage
 ^
 | ┌───┐     ┌───┐
 | │   │     │   │
 | │   └─────┘   └────>
 |________________________ Time


 It is either

High (1)

or

Low (0)

Nothing in between.

Why only two values?

Electronics can easily distinguish between

0 Volts  → Logic 0

5 Volts  → Logic 1

or in modern chips

0V

3.3V

1.8V

1.2V

depending on the technology.

This makes computers extremely reliable.


| Analog                       | Digital                        |
| ---------------------------- | ------------------------------ |
| Continuous values            | Discrete values                |
| Smooth signal                | Square wave                    |
| Infinite possible values     | Usually only 0 and 1           |
| More affected by noise       | Less affected by noise         |
| Harder to store/process      | Easy to store/process          |
| Used in microphones, sensors | Used in computers, smartphones |



Exam Notes
Analog signal: Continuous signal that can take any value within a range.
Digital signal: Discrete signal, usually represented by binary values (0 and 1).
Dynamic range: The minimum-to-maximum range of values an analog signal can take.
Analog circuit: Processes continuous signals (e.g., amplifiers).
Digital circuit: Processes binary signals using logic gates (e.g., computers, calculators
